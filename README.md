## 免费版够不够用？

说实话，免费版已经很强了——PHP 5.6 到 8.5、Node.js 12 到 24、Python 2.7 到 3.14、Go、Java、.NET、Ruby、Rust 全有，MySQL、MariaDB、PostgreSQL、MongoDB、Redis 也都支持。一个人做小项目，5 个站点基本够用。

但有几个场景免费版明显不够：

- **项目多了**：5 个站点的限制很快就触顶，一旦同时维护多个独立项目，就需要 Pro；
- **需要真实 SSL 证书**：免费版只能用私有 CA 的自签证书，不被浏览器信任。如果要内网穿透给他人访问，或测试 HTTPS 回调，就必须 Pro 的 ACME 自动证书；
- **本地测试邮件功能**：内置邮件服务器是 Pro 专属，免费版没有；
- **数据安全**：自动定时备份也是 Pro 功能，重要项目数据不备份总归不踏实。

简单来说：**个人轻度使用，免费版可以先用着看。项目一多、或者有 HTTPS / 邮件测试需求，Pro 是刚需。**
